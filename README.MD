# Anotações - Origamid

## HTML & CSS Insights

### Tag `<img>`

- Por padrão, a tag `<img>` é tratada como `display: inline-block`.

### Propriedade `margin`

- A propriedade `margin` refere-se ao espaçamento externo do **documento** e não está relacionada ao modo de escrita, `writing-mode: vertical-rl;`.
